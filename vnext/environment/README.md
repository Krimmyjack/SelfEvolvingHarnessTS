# vNext frozen environment

This directory is the only environment definition for M0 through Task G.  The
`pyproject.toml` pins the published v0.2 operator stack, the raw-data readers, and the
historically recorded Torch/Chronos versions.  A real `uv.lock` must be generated by the
pinned uv release and committed before M0 reproduction; absence of that lock is a hard
blocker, never an implicit request to use the host interpreter.

Deterministic commands run with:

```sh
PYTHONHASHSEED=20260713 \
OMP_NUM_THREADS=1 \
MKL_NUM_THREADS=1 \
TZ=UTC \
LC_ALL=C \
LANG=C \
uv run --frozen python -m SelfEvolvingHarnessTS.vnext m0-audit \
  --root SelfEvolvingHarnessTS
```

The environment is intentionally not considered frozen until `uv.lock` exists and
`EnvironmentProbeV1.lock_matches` is true.
